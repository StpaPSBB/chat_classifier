# Chat Message Classifier

## Описание проекта

Сервис для автоматического анализа тематического состава групповых чатов. Приложение принимает историю переписки в формате JSON, классифицирует каждое сообщение по шести категориям и возвращает визуализацию распределения в виде круговой диаграммы.

## Основная функция

Классификация сообщений чатов по категориям:
- Организация (собрания, дедлайны, планирование)
- Учёба (лекции, задания, учебные вопросы)
- Техническое (код, ошибки, технологии)
- Неформальное (приветствия, общение вне работы)
- Вакансии (работа, карьера, трудоустройство)
- Другое (все остальные сообщения)

## Технологический стек

**Бэкенд:**
- FastAPI (Python веб-фреймворк)
- OpenRouter API (доступ к нейросетевым моделям)
- DeepSeek R1 (модель классификации текста)
- Matplotlib (генерация диаграмм)

**Фронтенд:**
- React
- Axios для HTTP-запросов
- Drag-and-drop загрузка файлов

## Архитектурные особенности

1. **Модульная структура** — разделение на сервисы работы с файлами, нейросетью и графикой
2. **Streaming Response** — передача изображений через потоки байтов без сохранения на диск
3. **Автоматическая очистка** — временные файлы удаляются сразу после обработки
4. **Уникальные идентификаторы** — каждый запрос получает ID для отслеживания

## Формат данных

**Вход:** JSON файл со структурой:
```json
{ ...,
  "messages": [
    {...,"text": "Текст сообщения 1",...},
    {...,"text": "Текст сообщения 2",...}
  ]
}
```

**Выход:**
- PNG изображение с круговой диаграммой

## Процесс обработки

1. Загрузка и валидация JSON файла
2. Извлечение и очистка текстов сообщений
3. Классификация через нейросетевую модель
4. Генерация диаграммы распределения
5. Возврат результата с автоматической очисткой ресурсов

## Применение

- Анализ активности в учебных чатах
- Мониторинг тематики корпоративных переписок
- Исследование интересов в сообществах
- Получение объективной статистики по коммуникациям

## Преимущества

- Объективный анализ без субъективных оценок
- Визуальное представление результатов
- Простота использования
- Современный стек технологий
- Готовность к интеграции с другими системами
